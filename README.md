## Getting Started

Prerequisites
- Node.js (v14 or higher)
- npm

Installation
```bash
git clone https://github.com/shorty-huddybuddy/Portfolio.git
cd Portfolio
npm install
npm start
```

Build for production
```bash
npm run build
```

Deploy to Vercel (if you use Vercel)
```bash
vercel deploy
```
